Sobre mí

Soy un desarrollador de software apasionado por crear soluciones elegantes y funcionales. Mi enfoque se centra en escribir código limpio, mantenible y escalable, siempre buscando las mejores prácticas en cada proyecto.

Actualmente me encuentro en formación en Campuslands, donde he desarrollado proyectos que abarcan desde bases de datos relacionales (MySQL) y NoSQL (MongoDB), desarrollo backend con Java y Python, hasta interfaces de usuario modernas y responsivas con HTML, CSS y JavaScript.

Mi filosofía de trabajo se basa en tres pilares fundamentales: organización en cada etapa del desarrollo, información actualizada y relevante, y gestión eficiente del tiempo para cumplir objetivos sin sacrificar calidad.
